## 🚀 快速开始

### 克隆仓库

```bash
git clone https://github.com/lllray/awesome-skills.git
cd awesome-skills
```

### 使用Skills

1. 选择需要的Skill分类
2. 访问对应仓库
3. 按照README说明安装使用

### 在OpenClaw中使用

```bash
cp -r skills/* ~/.openclaw/skills/
openclaw gateway restart
```
